这个系统为自己的交易提供信息支撑，主要包括以下几个部分：
基本面分析提示
情绪面分析提示
交易大佬开单信息整合
k线交易信号提醒

不定期学习别人或者自己构思一些新的交易策略：
然后对往期的数据进行手动或者自动回测放在文件夹“策略中”，每个策略装在一个文件夹中，比如动量交易策略“moment_trading_stradge”就存放在./doc/moment_trading_stradge下面，分别存放以下内容
```
moment_trading_stradge.txt/py               txt的话主要是用pine语言进行编写，可在tradeveiw平台上进行自动化数据回测，如果是py结尾的话就是使用python进行编写的。
data                                        采集到的数据会被存放在该目录 
test_report                                 存放策略的测试包括，包括胜率，收益率以及最大回撤
```
